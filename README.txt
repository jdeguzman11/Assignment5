This assignment 5 adapts on the Profile.py file and ds_protocol.py file which extends to be able to interact with other users with a newly created GUI. Our code implements tkinter to create an interactable and user-freidnly GUI that allows users to load a DSU profile (through clotho.ics.uci.edu server and port 2021), add users with a valid profile, and send them messages. Our program stores messages and constanly refreshes each side to be able to have an up-to-date quick messaging system with one another.
